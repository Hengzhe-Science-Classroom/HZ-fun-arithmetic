# Fun Arithmetic & Number Sense

Interactive elementary math with visualizations — part of [Hengzhe's Science Classroom](https://hengzhe-science-classroom.github.io/hub/).

**[Open the App](https://hengzhe-science-classroom.github.io/HZ-fun-arithmetic/)**

## Chapters

| Ch | Topic |
|----|-------|
| 0 | Counting & Place Value |
| 1 | Addition Adventures |
| 2 | Subtraction Stories |
| 3 | Multiplication Magic |
| 4 | Division Discovery |
| 5 | Patterns & Sequences |
| 6 | Even & Odd Properties |
| 7 | Estimation & Rounding |
| 8 | Word Problems |
| 9 | Number Games |

## Features

- **10 chapters** of interactive arithmetic content
- **Visualizations** with draggable points, sliders, and animations
- **KaTeX** math rendering
- **Dark theme** with progress tracking (saved in localStorage)
- **Zero dependencies** — vanilla HTML/CSS/JS

## Quick Start

```bash
git clone https://github.com/Hengzhe-Science-Classroom/HZ-fun-arithmetic.git
cd HZ-fun-arithmetic
python3 -m http.server 8000
# Open http://localhost:8000
```

Or just visit the [live app](https://hengzhe-science-classroom.github.io/HZ-fun-arithmetic/).
